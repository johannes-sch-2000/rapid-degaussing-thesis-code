# Fieldmap_QA Script

## Overview
The Fieldmap_QA script is a comprehensive analysis and visualization tool for magnetic field measurements acquired with the Fieldmap GUI.

It enables:
- Time-domain inspection of signals
- Frequency-domain analysis (PSD, ASD, FFT)
- Spatial fieldmap evaluation
- Gradient and homogeneity analysis
- AUX channel diagnostics (including hysteresis behavior)
- Comparison between different fieldmaps

The script is designed for interactive use via a command-line interface.

---

## Required Input Data

The script operates on a **session folder** generated by the Fieldmap GUI. Typical contents:

- stream_f32.bin → main magnetic field channels
- stream_aux_f32.bin → auxiliary channels (e.g. voltage, current)
- summary.csv → processed fieldmap
- summary_raw.csv → raw fieldmap
- aux_points.csv → manually recorded points (optional)

---

## How to Run

Run the script from a terminal:

python Fieldmap_QA.py

You will be prompted to:
1. Select a session directory
2. Choose an analysis option

---

## Analysis Options

### Signal Analysis
1. Time trace → raw or corrected field signals
2. Chunk mean & standard deviation → stability over time
3. PSD + ASD → noise spectrum
4. Spectrogram → time-frequency behavior
5. Event detection → spikes or disturbances
6. Allan deviation → long-term stability

---

### Fieldmap Analysis
10. Fieldmap QA:
    - Mean field
    - Gradient metrics
    - RMS field variation

11. 3D fieldmap visualization:
    - Thesis-style plots
    - Fixed color scale (recommended for comparison)

12. Fieldmap comparison:
    - Difference between two measurements
    - Useful for detecting systematic errors

---

### AUX Analysis

13. AUX Points QA:
    - Field magnitude over recorded AUX points
    - Differences between consecutive points
    - Includes user comments

14. AUX XY / Hysteresis QA:
    - Plots one AUX channel against another
    - Detects signal onset automatically
    - Displays first N periods of excitation
    - Computes product statistics (X·Y)

Use this to verify:
- sufficient drive amplitude
- nonlinear response
- saturation behavior

---

## Important Settings (Option 6)

### General
- Axis selection (X/Y/Z)
- Offset correction mode
- Chunk duration

### Fieldmap
- Color scale limits (e.g. 0–12 nT)
- Cone scaling behavior

### AUX XY
- Channel selection (X/Y)
- Number of periods to display
- Trigger detection threshold
- Plot mode (line or scatter)

---

## Interpretation Guidelines

### Fieldmaps
- Low mean field → good shielding
- Low gradients → suitable for OPM operation

### AUX XY (Hysteresis)
- Wide loop → strong excitation
- Flattening (“beak”) → indication of saturation
- Loop evolution → dynamic material response

## Troubleshooting

Problem: No data found  
→ Check session folder structure

Problem: Noisy XY plot  
→ Increase trigger threshold

Problem: Plot too dense  
→ Reduce number of periods or increase downsampling
